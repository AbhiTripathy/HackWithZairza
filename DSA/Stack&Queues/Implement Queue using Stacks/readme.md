## Problem 2: Implement Queue using Stacks
Implement a FIFO queue using only two stacks. Support push, pop, peek, and empty operations.

### Example:
```
MyQueue queue = new MyQueue();
queue.push(1);
queue.push(2);
queue.peek();  // returns 1
queue.pop();   // returns 1
queue.empty(); // returns false
```
