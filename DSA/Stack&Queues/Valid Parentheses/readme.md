## Problem 1: Valid Parentheses
Given a string s containing just the characters '(', ')', '{', '}', '[', ']', determine if the input string is valid. Opening brackets must be closed by the same type in the correct order.

### Example:
```
Input: s = "()[]{}"
Output: true

Input: s = "([)]"
Output: false
```

### Constraints:
- `1 <= s.length <= 10^4`
